# แชร์ Agent Skills ที่ฉันสร้างไว้ — แหล่งรวม AI Coding Skills

บทความนี้เป็น **แหล่งรวม** Agent Skills ที่ผมสร้างและดูแลไว้สำหรับ AI Coding Assistant ครับ แต่ละ Skill เปรียบเสมือน "สมองเสริม" ที่ทำให้ AI เข้าใจ context เฉพาะทาง แล้วทำงานได้แม่นยำและมีคุณภาพกว่าปกติ

> 📌 **บทความนี้จะอัปเดตเรื่อยๆ** เมื่อมี Skill ใหม่ — กลับมาดูได้เสมอ!

---

## Agent Skill คืออะไร?

Agent Skill คือชุดคำสั่งและ context ที่ป้อนให้ AI Coding Assistant (เช่น Gemini, Copilot, Cursor) เพื่อให้มัน:

- **เข้าใจ design system** เฉพาะทาง เช่น สีที่ใช้, typography, spacing
- **สร้าง code ที่มีคุณภาพ** ตาม best practices ที่กำหนดไว้
- **ลดการแก้ไขซ้ำ** เพราะ AI รู้ "กฎ" ตั้งแต่ต้น

### โครงสร้างของ Skill

```
skills/
└── skill-name/
    ├── SKILL.md          ← คำสั่งหลัก (frontmatter + instructions)
    ├── scripts/          ← Helper scripts (ถ้ามี)
    ├── examples/         ← ตัวอย่างโค้ด
    ├── resources/        ← ไฟล์เสริม
    └── references/       ← เอกสารอ้างอิง
```

### Skill นี้ทำอะไร?

เป็น **masterclass-level reference** ที่ศึกษาจากเว็บ [tailwindcss.com](https://tailwindcss.com) แบบ pixel-perfect — อ่านตรง live source code จริง แล้วสรุปออกมาเป็นชุดกฎที่ AI สามารถนำไปใช้สร้าง UI ได้ทันที ครอบคลุม **11 หัวข้อ**:

1. **System Tokens & Color Palette** — ชุดสีที่ใช้จริง (เปลี่ยนจาก `slate-*` → `gray-*` ตาม v4)
2. **Tailwind v4 CSS-First Configuration** — `@theme`, `@utility`, `@custom-variant dark`
3. **Typography System** — Inter Variable + IBM Plex Mono, `text-lg/7` shorthand, `text-balance`
4. **Visual Hierarchy Techniques** — 200vw pseudo-element dividers, diagonal stripe gutter, CSS var opacity
5. **Grid & Layout System** — `grid-rows-[1fr_1px_auto]`, gutter columns, 3-column docs layout
6. **Glowing Mesh Grids & Spotlights** — Background grid, ambient glow, laser beams
7. **Gradient Border Glow Cards** — การ์ดที่เรืองแสงเมื่อ hover
8. **Responsive Design Patterns** — Dual-element pattern, `max-sm:hidden`, breakpoint range syntax
9. **Accessibility (A11y)** — Focus rings, `aria-hidden`, `aria-label`, `aria-haspopup`, skip-nav
10. **Button & Interactive Patterns** — CTA, Version badge, Plus badge, Search bar
11. **Dark Mode Implementation** — oklch theme-color, image swap, dark class strategy

### ตัวอย่างโค้ดจาก Skill

#### Full-Viewport Section Dividers (v4 Trick)

```html
<!-- เส้น hairline ที่พาด full browser width โดยไม่สนใจ max-width ของ content -->
<div class="relative
  before:absolute before:top-0 before:h-px before:w-[200vw] before:left-[-100vw]
  before:bg-gray-950/5 dark:before:bg-white/10
  after:absolute after:bottom-0 after:h-px after:w-[200vw] after:left-[-100vw]
  after:bg-gray-950/5 dark:after:bg-white/10">
  <!-- Section content -->
</div>
```

#### Hero Page Grid (Tailwind v4)

```html
<div class="grid min-h-dvh grid-cols-1
            grid-rows-[1fr_1px_auto_1px_auto]
            [--gutter-width:2.5rem]
            md:grid-cols-[var(--gutter-width)_minmax(0,var(--breakpoint-2xl))_var(--gutter-width)]">
  <!-- 1px rows = automatic dividers, no extra HTML needed -->
</div>
```

#### Gradient Border Glow Card (v4 colors)

```html
<div class="relative group overflow-hidden rounded-2xl
  border border-gray-950/5 bg-white p-6 transition-all duration-300
  dark:border-white/10 dark:bg-gray-900/60 hover:-translate-y-0.5">

  <!-- Glowing gradient on hover -->
  <div class="absolute -inset-px -z-10 rounded-2xl bg-gradient-to-r
    from-sky-500 to-indigo-500 opacity-0 blur transition-all duration-300
    group-hover:opacity-20 group-hover:blur-md">
  </div>

  <h3 class="text-lg font-semibold text-gray-950 dark:text-white mb-2">Card Title</h3>
  <p class="text-sm text-gray-500 dark:text-gray-400">Card content text.</p>
</div>
```

#### Accessible Focus Ring

```html
<button class="focus:outline-none
               focus-visible:ring-2
               focus-visible:ring-sky-500
               focus-visible:ring-offset-2
               dark:focus-visible:ring-offset-gray-950">
  Click me
</button>
```

### Color Palette Reference (v4 — gray-* ไม่ใช่ slate-*)

| Role | Light Mode | Dark Mode | หมายเหตุ |
|------|-----------|-----------|----------|
| **Background** | `bg-white` | `dark:bg-gray-950` | oklch(.13 .028 261) |
| **Surface** | `bg-gray-50` | `dark:bg-gray-900` | panels/cards |
| **Subtle Border** | `border-gray-950/5` | `dark:border-white/10` | opacity-based |
| **Divider Line** | `bg-gray-950/5` | `dark:bg-white/10` | pseudo-elements |
| **Primary Text** | `text-gray-950` | `dark:text-white` | headings |
| **Body Text** | `text-gray-600` | `dark:text-gray-400` | paragraphs |
| **Brand Sky** | `text-sky-500` | `dark:text-sky-400` | `#0ea5e9` |
| **Monospace Label** | `text-sky-500` | `dark:text-sky-400` | badges, code |

### วิธีติดตั้ง

```bash
# Clone repo
git clone https://github.com/vick42uho/tailwind-ux-ui.git

# คัดลอกไปยัง skills directory
cp -r tailwind-ux-ui ~/.gemini/config/skills/
```

หรือจะสร้าง `skills.json` เพื่อ reference จาก path ที่ต้องการ:

```json
{
  "entries": [
    { "path": "path/to/tailwind-ux-ui" }
  ]
}
```
