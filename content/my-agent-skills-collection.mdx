# แชร์ Agent Skills ที่ฉันสร้างไว้ — แหล่งรวม AI Coding Skills

บทความนี้เป็น **แหล่งรวม** Agent Skills ที่ผมสร้างและดูแลไว้สำหรับ AI Coding Assistant ครับ แต่ละ Skill เปรียบเสมือน "สมองเสริม" ที่ทำให้ AI เข้าใจ context เฉพาะทาง แล้วทำงานได้แม่นยำและมีคุณภาพกว่าปกติ

> 📌 **บทความนี้จะอัปเดตเรื่อยๆ** เมื่อมี Skill ใหม่ — กลับมาดูได้เสมอ!

---

## Agent Skill คืออะไร?

Agent Skill คือชุดคำสั่งและ context ที่ป้อนให้ AI Coding Assistant (เช่น Gemini, Copilot, Cursor) เพื่อให้มัน:

- **เข้าใจ design system** เฉพาะทาง เช่น สีที่ใช้, typography, spacing
- **สร้าง code ที่มีคุณภาพ** ตาม best practices ที่กำหนดไว้
- **ลดการแก้ไขซ้ำ** เพราะ AI รู้ "กฎ" ตั้งแต่ต้น

### โครงสร้างของ Skill

```
skills/
└── skill-name/
    ├── SKILL.md          ← คำสั่งหลัก (frontmatter + instructions)
    ├── scripts/          ← Helper scripts (ถ้ามี)
    ├── examples/         ← ตัวอย่างโค้ด
    ├── resources/        ← ไฟล์เสริม
    └── references/       ← เอกสารอ้างอิง
```

### Skill นี้ทำอะไร?

เป็น **masterclass-level reference** ที่ศึกษาจากเว็บ [tailwindcss.com](https://tailwindcss.com) แบบ pixel-perfect แล้วสรุปออกมาเป็นชุดกฎที่ AI สามารถนำไปใช้สร้าง UI ได้ทันที ครอบคลุม:

1. **System Tokens & Color Codes** — ชุดสีที่ใช้จริง ไม่ใช่สีกว้างๆ
2. **Tailwind CSS v4 Configuration** — CSS-first config แบบใหม่
3. **Glowing Mesh Grids** — Background grid แบบเรืองแสง
4. **Ambient Spotlights & Beams** — แสง glow และเลเซอร์ตกแต่ง
5. **Gradient Border Glow Cards** — การ์ดที่เรืองแสงเมื่อ hover
6. **Typography Grid & Hierarchy** — กฎ typography ระดับ premium
7. **Interactive States** — Focus, Selection, Scrollbar styling
8. **Layout Systems** — Three-column documentation layout
9. **Code Editor Components** — Tab + code panel แบบเว็บ Tailwind
10. **Common Pitfalls** — สิ่งที่ต้องหลีกเลี่ยง

### ตัวอย่างโค้ดจาก Skill

#### Dark Mode Glowing Grid Background

```html
<div class="absolute inset-0 -z-10 h-full w-full bg-slate-950
  bg-[linear-gradient(to_right,#1e293b_1px,transparent_1px),linear-gradient(to_bottom,#1e293b_1px,transparent_1px)]
  bg-[size:4rem_4rem]
  [mask-image:radial-gradient(ellipse_60%_50%_at_50%_0%,#000_70%,transparent_100%)]
  opacity-80">
</div>
```

#### Gradient Border Glow Card

```html
<div class="relative group overflow-hidden rounded-2xl border border-slate-200/80
  bg-white p-6 transition-all duration-300
  dark:border-slate-800 dark:bg-slate-900/60 hover:-translate-y-0.5">

  <!-- Glowing gradient on hover -->
  <div class="absolute -inset-px -z-10 rounded-2xl bg-gradient-to-r
    from-sky-500 to-indigo-500 opacity-0 blur transition-all duration-300
    group-hover:opacity-20 group-hover:blur-md">
  </div>

  <!-- Subtle border highlight -->
  <div class="absolute inset-[1px] rounded-[15px]
    bg-gradient-to-b from-white/10 to-transparent pointer-events-none">
  </div>

  <!-- Content -->
  <h3 class="text-lg font-semibold text-slate-900 dark:text-white mb-2">Card Title</h3>
  <p class="text-sm text-slate-500 dark:text-slate-400">Card content text.</p>
</div>
```

### วิธีติดตั้ง

```bash
# Clone repo
git clone https://github.com/vick42uho/tailwind-ux-ui.git

# คัดลอกไปยัง skills directory
cp -r tailwind-ux-ui ~/.gemini/config/skills/
```

หรือจะสร้าง `skills.json` เพื่อ reference จาก path ที่ต้องการ:

```json
{
  "entries": [
    { "path": "path/to/tailwind-ux-ui" }
  ]
}
```
