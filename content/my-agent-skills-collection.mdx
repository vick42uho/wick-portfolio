## ทำไมต้องสร้าง Agent Skill?

Skills คือวิธีที่ดีที่สุดในการยกระดับ AI Coding Assistant ให้เข้าใจ context เฉพาะทาง:

- ✅ **ลดเวลาแก้ไข** — AI ทำถูกตั้งแต่ครั้งแรก
- ✅ **Consistency** — ทุกโปรเจกต์ใช้ patterns เดียวกัน
- ✅ **Reusable** — สร้างครั้งเดียว ใช้ได้ทุกโปรเจกต์
- ✅ **Shareable** — แชร์ให้ทีมหรือ community ได้
- ✅ **Living Documentation** — Skill คือ documentation ที่ AI อ่านแล้วนำไปใช้ได้เลย

> 💡 **มี Skill ที่ใช้อยู่?** หรืออยากให้สร้าง Skill ใหม่ — ติดต่อมาได้ครับ!
