# Antigravity Rules for my-portfolio

## Project Overview
This is a personal portfolio and blog built with **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, and **Content Collections (MDX)**.

## Core Rules & Guidelines
1. **MDX Content**:
   - Store all blog posts in `content/*.mdx`.
   - Ensure every `.mdx` file includes valid frontmatter (`title`, `publishedAt`, `updatedAt`, `author`, `summary`, `image`).
   - Use Thai language for blog post content unless specified otherwise.
2. **Code Quality**:
   - Use TypeScript strict mode and proper type definitions.
   - Verify code using `npx next build` or `pnpm build` before declaring success.
3. **Design System**:
   - Follow dark mode theme and clean modern UI aesthetics.
   - Maintain responsiveness across mobile and desktop breakpoints.
